ZCam is a CAM (Computer Aided Manufacturing) program. It interactively processes CAD data
and generates control data for CNC machines or lasers.
ZCam can both generate these control data (G-Code) and directly control some lasers (Galvo lasers).

ZCam is an attempt to re-implement a predecessor based on an outdated platform.

The current state is incomplete and under development.

**This is experimental work in progress**
