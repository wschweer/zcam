# ZCam

ZCam is a CAM (Computer Aided Manufacturing) program. It allows you to import and edit CAD data. ZCam generates control data for Galvo fiber lasers or G-code for CNC machines.

[<img src="manual/foto-1.png" style="width:300px;"/>](manual/foto-1.png)

ZCam is based on an older (unpublished) program of mine. This is an attempt to port that program to a more modern platform.

In its current state, ZCam is still under development and is not yet usable.
